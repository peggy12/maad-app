Step 1: Create Railway Account

Go to https://railway.app and sign up for free

Step 2: Deploy from GitHub

1. Click "Deploy from GitHub repo"

2. Connect your GitHub account

3. Select your maad-app repository

4. Railway will auto-detect it's a Node.js app

Step 3: Configure Environment

In Railway dashboard, set these environment variables:

NODE_ENV=production
PORT=3000

Step 4: Update package.json

Make sure your package.json has:

{
"scripts": {
"start": "node railway-webhook.js"
},
"dependencies": {
"express": "^4.18.2"
}
}

Step 5: Get Your Webhook URL

After deployment, Railway will give you a URL like:

https://your-app-name.railway.app/webhook

Use this URL in Facebook Developer Console with verify token: maadlad_verify_token

Alternative: Use webhook.site for immediate testing

For immediate testing while setting up Railway:

  1. Go to webhook.site
  2. Copy your unique URL
  3. Use it in Facebook Developer Console (no verify token needed)
  4. You'll see all webhook events in real-time
